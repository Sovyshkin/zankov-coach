# SEO Performance Monitoring Guide for ZankovCoach

## Основные инструменты для мониторинга SEO

### 1. Google Search Console
- **URL**: https://search.google.com/search-console
- **Что отслеживать**:
  - Позиции по ключевым словам
  - Количество кликов и показов
  - CTR (кликабельность)
  - Ошибки индексации
  - Производительность мобильной версии

### 2. Google Analytics 4
- **URL**: https://analytics.google.com
- **Ключевые метрики**:
  - Органический трафик
  - Время на сайте
  - Показатель отказов
  - Конверсии (заявки на тренировки)
  - География посетителей

### 3. Яндекс.Метрика
- **URL**: https://metrica.yandex.ru
- **Для российской аудитории**:
  - Карты кликов
  - Записи сеансов
  - Конверсии
  - Поисковые фразы

## Технические метрики для мониторинга

### Скорость загрузки:
- **Инструменты**: PageSpeed Insights, GTmetrix
- **Цель**: Скорость загрузки < 3 секунд
- **Мобильная версия**: > 90 баллов в PageSpeed

### Core Web Vitals:
- **LCP** (Largest Contentful Paint): < 2.5s
- **FID** (First Input Delay): < 100ms
- **CLS** (Cumulative Layout Shift): < 0.1

### Индексация:
- Проверка индексации всех страниц
- Мониторинг sitemap.xml
- Отслеживание ошибок 404
