# Инструкция по добавлению портрета тренера

## Где разместить изображение

Поместите фотографию тренера в папку `public/` с именем `trainer-portrait.jpg`

**Путь к файлу:** `/public/trainer-portrait.jpg`

## Требования к изображению

### Размер и пропорции:
- **Рекомендуемый размер:** 800x1000 пикселей (соотношение 4:5)
- **Минимальный размер:** 600x750 пикселей
- **Максимальный размер:** 1200x1500 пикселей

### Качество:
- **Формат:** JPG или PNG
- **Качество:** Высокое разрешение (минимум 72 DPI, лучше 150+ DPI)
- **Размер файла:** До 2 МБ

### Композиция:
- **Ориентация:** Портретная (вертикальная)
- **Кадрирование:** От груди до головы или в полный рост
- **Фон:** Желательно нейтральный или размытый
- **Освещение:** Хорошее, равномерное освещение лица

## Стилизация

Изображение будет автоматически стилизовано следующим образом:
- Скругленные углы (20px)
- Зеленая рамка с эффектом свечения
- Легкая тень для глубины
- Небольшое увеличение при наведении
- Минимальный эффект обесцвечивания для стильности

## Fallback

Если файл `trainer-portrait.jpg` не найден, будет показан placeholder изображение фитнес-тренера из Unsplash.

## Альтернативные имена файлов

Если хотите использовать другое имя файла, измените путь в файле:
`/src/components/HeroSection.vue` в строке:
```html
src="/trainer-portrait.jpg"
```

Замените на ваше имя файла, например:
```html
src="/vladislav-photo.jpg"
```

## Проверка

После добавления изображения:
1. Убедитесь, что файл находится в папке `public/`
2. Перезагрузите страницу
3. Изображение должно появиться в левой части первого блока
4. При наведении должен появляться эффект увеличения и свечения
